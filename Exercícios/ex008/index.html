Nessa frase temos um termo em Destaque usando a tag <strong>(semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em Ênfase usando a tag <em> (semântica).

Texto marcado

Podemos também criar um texto marcado utilizando a tag <mark>

Podemos mudar a cor de um termo marcado alterando os parâmetros de cor da tag <mark>

Se mudarmos o parâmetros do cabeçalho, podemos mudar a configuração de <mark> para o site todo, basta utilizar: 


    <style>
        mark {
        background-color: aquamarine;
        }
    </style>

Assim, todos os termos marcados terão as mesmas cores.

Texto Deletado

Podemos marcar um texto como excluído, para que seja lido, mas não considerado com a tag <del>

Texto Inserido

Também se pode marcar um texto como inserido, para que fique sublinhado, de forma a preencher, com a tag <ins> (semântica).

Para apenas sublinhar o texto, utiliza-se a tag <u> (não semântica).
